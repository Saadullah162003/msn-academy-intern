Course Overview

This course builds a strong foundation in algorithmic thinking and competitive programming. You’ll cover time–space complexity, data structures, classical algorithms, and contest strategies. Learn patterns like two-pointers, binary search, greedy, dynamic programming, graphs, trees, and number theory while solving curated problems from easy to advanced.

Learning Outcomes
  • ✔ Analyze problems and design optimal algorithms (Big-O mastery)
  • ✔ Implement core data structures: stacks, queues, heaps, hash maps, DSU
  • ✔ Solve DP (1D/2D, knapsack, LIS, bitmask, digit DP) efficiently
  • ✔ Work with graphs: BFS/DFS, shortest paths (Dijkstra), MST (Kruskal/Prim), topo sort
  • ✔ Apply number theory: primes, sieve, gcd/lcm, modular arithmetic, fast exponentiation
  • ✔ Build segment trees & Fenwick trees (range queries/updates)
  • ✔ Practice contest strategy: reading, debugging fast, partial scoring
Target Audience

Ideal for students preparing for coding competitions (ICPC, Codeforces, LeetCode, Hackerrank), interviews, or anyone who wants to sharpen problem-solving skills. Basic programming knowledge required.
